# Programming on SQL Server

Repository dedicated to practicing SQL Server and T-SQL concepts through progressive scripts organized by database domain.

The project is divided into two datasets — **Company** and **Nutrition** — and applies the same sequence of SQL topics in both contexts, reinforcing learning through repetition and variation of scenarios.

## Topics covered

- Data retrieval
- Filtering and ordering
- Multi-table queries
- Grouping and aggregation
- Views
- Subqueries
- Scalar functions
- Table-valued functions

## Project structure

Each folder contains a database setup script and a sequence of SQL exercises covering the same progression of topics.

- `database-company/`
- `database-nutrition/`

## Purpose

This repository was created as a study project to strengthen practical knowledge of SQL Server, database querying, and T-SQL programming.
